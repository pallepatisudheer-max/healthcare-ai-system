# 🏥 AI-Powered Healthcare Management System

An intelligent healthcare platform built using **Python**, **Streamlit**, and **Google Gemini AI** that helps hospitals manage patients, doctors, appointments, healthcare resources, and AI-powered medical assistance.

---

## 🚀 Features

### 👨 Patient Management
- Add, update, delete, and view patient records
- Maintain patient information efficiently

### 👨‍⚕️ Doctor Management
- Manage doctor details
- Track specialties and availability

### 📅 Appointment Scheduling
- Book and manage appointments
- Organize patient-doctor interactions

### 🧠 Disease Prediction
- Machine Learning-based disease prediction
- Predict potential health conditions using patient data

### 💊 Treatment Recommendation
- AI-powered treatment suggestions
- Healthcare guidance and recommendations

### 🤖 AI Healthcare Assistant
- Powered by Google Gemini AI
- Answers healthcare-related questions
- Provides educational medical information

### 🛏 Bed Management
- Track available and occupied hospital beds
- Improve hospital resource utilization

### 👩‍⚕️ Staff Scheduling
- Manage healthcare staff schedules
- Improve workforce planning

### 📈 Analytics Dashboard
- Visual healthcare insights
- Key performance indicators and statistics

---

## 🛠️ Technologies Used
